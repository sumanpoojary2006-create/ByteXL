# Unit 2 Visual Blueprint

## Theme

Python Workshop Counter: data types and operators are shown as stations on a clean workshop counter. Values arrive as physical items, variables store them in labelled drawers, type conversion uses an adapter, operators act like tools, comparisons and logic use everyday switches, and output is shown as a receipt or screen.

## Style Guidelines

Clean cropped-wide 3D-style raster PNG infographic, light blue-white background, softly shaded objects, rounded raised cards, subtle shadows, bold readable sans-serif labels, and a limited accent palette of blue, teal, green, orange, purple, red, and slate. Do not add top title or subtitle headers inside the image because the reading material already provides them; keep only concise labels that identify objects, values, and actions.
