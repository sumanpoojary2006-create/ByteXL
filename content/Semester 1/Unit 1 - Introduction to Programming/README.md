# Unit 1: Introduction to Programming

**Semester 1: Python Fundamentals**

Goal of this unit: think like a programmer before writing Python, turning real problems into clear logic.

Each lesson is written as a short, story-led read rather than a fixed template. The flow is consistent: a standardized **Introduction**, the core idea explained through real-world examples and situation-based questions under natural headings, concept-scoped illustrations, and a closing **Conclusion**. Programming concepts are taught mostly through plain-English steps; actual Python code is introduced only in the final lesson, once the environment is set up.
