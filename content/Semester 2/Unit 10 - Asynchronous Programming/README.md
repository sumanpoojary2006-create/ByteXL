# Unit 10: Asynchronous Programming

**Semester 2: Advanced Python Concepts**

**Character thread:** Miguel, a backend developer whose sequential API calls take 3 seconds when they could take 300ms in parallel -- and who learns exactly when async helps and when it does not.
